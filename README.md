# Gym Management System

This is a **Gym Management System** web application developed using **Flask (Python)**.  
It provides basic information about the gym, membership plans, and other pages.

## Features

- **Home Page (`/`)** – Welcome page about the gym.  
- **About Page (`/about`)** – Details about the gym, trainers, and facilities.  
- **Membership Page (`/membership`)** – Membership plans:  
  - **Basic Plan:** Gym floor access, locker, 1 trainer session/week  
  - **Pro Plan:** All Basic features + unlimited trainer sessions, diet & workout plan  
  - **Elite Plan:** All Pro features + 24/7 access, personal trainer & nutritionist  

## Technologies Used

- Python 3.12  
- Flask Web Framework  
- HTML/CSS  
- `templates/` folder for HTML templates  
- `static/` folder for CSS and images
